**日期**: 2026-04-03

### 数据集

| 数据集 | 路径 | 条数 | 说明 |
|--------|------|------|------|
| cleaned | `datasets/data/train/sft_trajectories_cleaned.jsonl` | 1583 | 从 MCTS 输出中筛选 reward > 0 的 best 轨迹，每场景最多 1 条 |

数据字段：`messages`（system/user/assistant/tool 多轮对话）, `reward`, `env_sample_idx`, `question`

数据来源链路：
```
collected_featurize_8c16g_*.json (场景)
  → MCTS run_search.py (采样)
    → sft_trajectories.jsonl (7791条, 全量)
      → sft_trajectories_cleaned.jsonl (1583条, reward>0 的 best 轨迹)
```

### 训练命令

```bash
cd ~/Compiler-R1/project/db-opt-r1

python -m training.trl.sft \
    --model_path ~/Qwen3-4B \
    --data_files datasets/data/train/sft_trajectories_cleaned.jsonl \
    --output_dir ./model_save/sft_qwen3_4b_cleaned/ \
    --num_epochs 3 \
    --lr 1e-5 \
    --batch_size 2 \
    --grad_accum 4 \
    --lora_rank 64 \
    --max_length 8192 \
    --flash_attn
```

- cleaned 数据集只保留了 reward > 0 的 best 轨迹（每个场景最多 1 条），质量较高
- 如显存不足，改 `--batch_size 1 --grad_accum 8`
- numpy 需锁定 `<2.0`（`pip install numpy==1.26.4`），trl 会拉高版本导致兼容问题
