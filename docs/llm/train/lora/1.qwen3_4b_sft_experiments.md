### 训练命令

```bash

    --model_path ~/Qwen3-4B \
    --data_files data_pipeline/data/train/sft_trajectories_cleaned.jsonl \
```

### LoRA 合并

```bash
python -c "
import os
from peft import PeftModel
from transformers import AutoModelForCausalLM, AutoTokenizer
import torch

base = os.path.expanduser('~/Qwen3-4B')
adapter = './model_save/sft_qwen3_4b_cleaned/'
output = './model_save/sft_qwen3_4b_cleaned_merged/'

model = AutoModelForCausalLM.from_pretrained(base, torch_dtype=torch.bfloat16, device_map='cpu')
tokenizer = AutoTokenizer.from_pretrained(base)
model = PeftModel.from_pretrained(model, adapter)
model = model.merge_and_unload()
model.save_pretrained(output)
tokenizer.save_pretrained(output)
print('done')
"
```

### 增量数据提取

从本次 eval + 训练轨迹中提取 LLM 预测的 knob 配置，用于扩充 cost model 训练数据。

**数据来源**：

| 文件 | 条数 | 说明 |
|------|------|------|
| `eval_results/sft_qwen3_4b_cleaned/sft_trajectories.jsonl` | ~578 | 本次 SFT 模型 eval 输出 |
| `data_pipeline/data/train/sft_trajectories_cleaned.jsonl` | 1583 | GPT-5 训练轨迹 |

**提取命令**：
```bash
python3 -m data_pipeline.synthesis.trajectory.extract_knobs \
    --trajectories eval_results/sft_qwen3_4b_cleaned/sft_trajectories.jsonl \
                   data_pipeline/data/train/sft_trajectories_cleaned.jsonl \
    --scenarios data_pipeline/data/scenarios/collected/ \
    --output data_pipeline/data/scenarios/knobs/knob_configs_incremental_v1.json
```

**结果**：2176 条轨迹 → 2074 条去重 knob 配置
