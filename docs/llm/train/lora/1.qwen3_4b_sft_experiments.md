### 训练命令

```bash
cd ~/Compiler-R1/project/db-opt-r1

python -m training.trl.sft \
    --model_path ~/Qwen3-4B \
    --data_files data_pipeline/data/train/sft_trajectories_cleaned.jsonl \
    --output_dir ./model_save/sft_qwen3_4b_cleaned/ \
    --num_epochs 3 \
    --lr 1e-5 \
    --batch_size 2 \
    --grad_accum 4 \
    --lora_rank 64 \
    --max_length 8192 \
    --flash_attn
```

### 训练结果

| 指标 | 值 |
|------|------|
| 模型路径 | `./model_save/sft_qwen3_4b_cleaned/` |

### 备注

- cleaned 数据集只保留了 reward > 0 的 best 轨迹（每个场景最多 1 条），质量较高
- 如显存不足，改 `--batch_size 1 --grad_accum 8`
- numpy 需锁定 `<2.0`（`pip install numpy==1.26.4`），trl 会拉高版本导致兼容问题

### LoRA 合并

```bash
python -c "
import os
from peft import PeftModel
from transformers import AutoModelForCausalLM, AutoTokenizer
import torch

base = os.path.expanduser('~/Qwen3-4B')
adapter = './model_save/sft_qwen3_4b_cleaned/'
output = './model_save/sft_qwen3_4b_cleaned_merged/'

model = AutoModelForCausalLM.from_pretrained(base, torch_dtype=torch.bfloat16, device_map='cpu')
tokenizer = AutoTokenizer.from_pretrained(base)
model = PeftModel.from_pretrained(model, adapter)
model = model.merge_and_unload()
model.save_pretrained(output)
tokenizer.save_pretrained(output)
print('done')
"
```

### Exp-2: Qwen3-4B Full SFT（verl）

**日期**: 2026-04-10

这次重新评估使用的是 **693** 场景的 eval 集，与上面的 GPT-5 / LoRA SFT 对照组保持同一批样本；评估报表使用 `--skip-bo`，因此不再统计 BO baseline 和 gap closed。

#### Full SFT 核心指标（双 baseline 体系）

| 指标 | 值 |
|------|------|
| 评估场景数 | 693 |
| 平均提升（vs 场景原始配置） | 14.57% |
| 中位数提升（vs 场景原始配置） | 3.24% |
| 提升 > 0% 比例（vs 场景原始配置） | 61.2% |
| 平均提升（vs PG 默认配置） | 109.85% |
| 中位数提升（vs PG 默认配置） | 97.15% |
| 提升 > 0% 比例（vs PG 默认配置） | 97.4% |
| 平均步数 | 10.0 |
| predict 调用率 | 100.0% |

#### Full SFT 结果解读

- 相比场景原始配置，**平均提升 14.57%**，已经明显高于前面的 LoRA SFT（13.32%，同为 693 场景评估）。
- 相比 PG 默认配置，**平均提升 109.85%**，说明 Full SFT 仍然能稳定给出远好于默认配置的参数组合。
- 平均步数 **10.0**，`predict` 调用率 **100%**，说明当前策略依然是“多轮探索 + predict 收尾”的模式，后续 GRPO 重点还是减少无效轮次、提升最终 answer 命中率。

#### 与已有结果对照

| 模型 | 评估场景数 | avg_imp_vs_scenario_pct | 备注 |
|------|-----------|-------------------------|------|
| GPT-5（教师模型） | 693 | 21.39% | 旧评估集 |
| LoRA SFT Qwen3-4B | 693 | 13.32% | 旧评估集 |
| Full SFT Qwen3-4B | 693 | 14.57% | 2026-04-10 复评，`--skip-bo` |
| Qwen3-4B baseline（未微调） | 484 | 0.0% | 旧评估集 |

> 结论：在与 GPT-5 / LoRA SFT 对齐的 693 场景评估集上，Full SFT Qwen3-4B 已经把场景级平均提升拉到 **14.57%**，高于 LoRA SFT 的 **13.32%**，说明这版 Full SFT 作为 GRPO 起点是成立的。
