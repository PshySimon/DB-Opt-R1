# 工具框架设计（修订版）

## 概述

每个工具是一个类，内含 `execute_real()` 和 `execute_simulated()` 两个方法，`execute()` 根据模式自动分发。

## 架构

```
Tool（Agent-R1 基类）
  │
  └── DBTool（DB 工具基类，扩展 mode + env_state）
        │
        ├── GetHardwareInfoTool
        │     ├── execute_real()       → 读 /proc/cpuinfo
        │     └── execute_simulated()  → 从 env_state 读
        │
        ├── SetKnobTool
        │     ├── execute_real()       → ALTER SYSTEM SET
        │     └── execute_simulated()  → 改 env_state
        │
        └── ... 共 11 个
```

## DBTool 基类

```python
class DBTool(Tool):
    """DB 工具基类，扩展 real/simulated 双模式"""

    def __init__(self, name, description, parameters,
                 mode="train", config=None, env_state=None):
        super().__init__(name, description, parameters)
        self.mode = mode
        self.config = config          # real 模式用
        self.env_state = env_state    # simulated 模式用

    def execute(self, args):
        if self.mode == "real":
            return self.execute_real(args)
        else:
            return self.execute_simulated(args)

    def execute_real(self, args) -> str:
        raise NotImplementedError

    def execute_simulated(self, args) -> str:
        raise NotImplementedError
```

## 工具实现示例

```python
class GetDBMetricsTool(DBTool):
    def __init__(self, **kwargs):
        super().__init__(
            name="get_db_metrics",
            description="获取 DB 运行时指标",
            parameters={"type": "object", "properties": {}, "required": []},
            **kwargs
        )

    def execute_real(self, args):
        conn = self.config.get_db_connection()
        cursor = conn.cursor()
        cursor.execute("SELECT sum(blks_hit), sum(blks_read) FROM pg_stat_database")
        ...
        return json.dumps(metrics)

    def execute_simulated(self, args):
        return json.dumps({
            k.replace("metric_", ""): v
            for k, v in self.env_state.items()
            if k.startswith("metric_")
        })


class SetKnobTool(DBTool):
    def __init__(self, **kwargs):
        super().__init__(
            name="set_knob",
            description="设置 PostgreSQL 参数",
            parameters={...},
            **kwargs
        )

    def execute_real(self, args):
        knobs = json.loads(args["knobs"])
        conn = self.config.get_db_connection()
        cursor = conn.cursor()
        for name, val in knobs.items():
            cursor.execute(f"ALTER SYSTEM SET {name} = %s", (str(val),))
        ...

    def execute_simulated(self, args):
        knobs = json.loads(args["knobs"])
        for k, v in knobs.items():
            self.env_state[f"knob_{k}"] = v
        return json.dumps({"success": list(knobs.keys())})
```

## 共享环境状态

Simulated 模式下，所有工具共享同一个 `env_state` 字典：

```python
env_state = {}

tools = [
    GetHardwareInfoTool(mode="train", env_state=env_state),
    SetKnobTool(mode="train", env_state=env_state),
    PredictPerformanceTool(mode="train", env_state=env_state, cost_model=model),
    ...
]
```

`reset()` 时从 CSV 加载一条样本填入 `env_state`，所有工具自动看到新数据。

## DBToolEnv

```python
class DBToolEnv(ToolEnv):
    def __init__(self, mode="train", config=None,
                 dataset_path=None, cost_model=None, max_turns=10):
        self.mode = mode
        self.env_state = {}

        common = {"mode": mode, "config": config, "env_state": self.env_state}
        tools = [
            GetHardwareInfoTool(**common),
            GetCurrentConfigTool(**common),
            GetDBMetricsTool(**common),
            GetWorkloadInfoTool(**common),
            GetRecentLogsTool(**common),
            SetKnobTool(**common),
            RestartPGTool(**common),
            ReloadPGTool(**common),
            ResetConfigTool(**common),
        ]

        if mode == "train":
            self.dataset = pd.read_csv(dataset_path)
            tools.append(PredictPerformanceTool(**common, cost_model=cost_model))
        else:
            tools.append(RunBenchmarkTool(**common))

        super().__init__(tools=tools, max_turns=max_turns)

    def reset(self, sample_idx=None):
        super().reset()
        if self.mode == "train":
            row = self.dataset.iloc[sample_idx or random.randint(...)]
            self.env_state.clear()
            self.env_state.update(row.to_dict())
```

## 11 个工具完整列表

| 工具类 | real 模式 | simulated 模式 |
|--------|----------|---------------|
| `GetHardwareInfoTool` | /proc/cpuinfo, /proc/meminfo | env_state[hw_*] |
| `GetCurrentConfigTool` | SHOW ALL | env_state[knob_*] |
| `GetDBMetricsTool` | pg_stat 视图 | env_state[metric_*] |
| `GetWorkloadInfoTool` | pg_database_size 等 | env_state[sw_*] |
| `GetRecentLogsTool` | tail PG 日志 | 根据 metrics 模拟生成 |
| `SetKnobTool` | ALTER SYSTEM SET | 改 env_state |
| `RestartPGTool` | pg_ctl restart | 返回 success |
| `ReloadPGTool` | pg_reload_conf() | 返回 success |
| `ResetConfigTool` | ALTER SYSTEM RESET ALL | 恢复原始 env_state |
| `PredictPerformanceTool` | — | Cost Model 预测 |
| `RunBenchmarkTool` | pgbench | — |
