# GRPO 训练 Pipeline 设计

## 概述

GRPO（Group Relative Policy Optimization）是 SFT 之后的强化学习阶段。模型在模拟环境中自主探索 DB 调优策略，通过 Cost Model 评估性能作为 reward 信号，持续优化策略。

**核心约束**：`db-opt-r1` 是独立项目，不能 import `agent_r1`。需要将 Agent-R1 的通用框架代码搬运过来，只保留必要模块，修改 import 路径，并替换任务特定代码。

## Reward 设计

```python
# training/reward_score.py

def compute_score_format(solution_str: str) -> float:
    """格式正确性评分（0~1.5）"""
    score = 0.0
    # 检查 <think> 标签
    if "<think>" in solution_str and "</think>" in solution_str:
        score += 0.5
    # 检查 <tool_call> 格式
    if "<tool_call>" in solution_str:
        score += 0.5
    # 检查轮次交替（user→assistant→tool→assistant...）
    if check_turn_alternation(solution_str):
        score += 0.5
    return score

def compute_score_answer(solution_str: str, ground_truth: dict) -> float:
    """任务完成度评分：提取 knob → Cost Model 预测 → 计算改善比例"""
    knobs = extract_final_knobs(solution_str)
    if not knobs:
        return 0.0
    predicted_tps = cost_model.predict(knobs, ground_truth["hardware"])
    baseline_tps = ground_truth["baseline_tps"]
    improvement = (predicted_tps - baseline_tps) / baseline_tps
    return max(0.0, min(1.0, improvement * 5))
```

## DB 工具适配

现有 `environment/tools/db_tools.py` 的工具是函数式的，需要包装为 `Tool` 子类：

```python
# 示例：将现有 set_knob 函数包装为 Tool
from training.tool.tool_base import Tool

class SetKnobTool(Tool):
    def __init__(self, configurator):
        super().__init__(
            name="set_knob",
            description="设置数据库配置参数",
            parameters={
                "type": "object",
                "properties": {
                    "knob_name": {"type": "string", "description": "参数名"},
                    "value": {"type": "string", "description": "参数值"},
                },
                "required": ["knob_name", "value"]
            }
        )
        self.configurator = configurator

    def execute(self, args: dict) -> str:
        return self.configurator.set_knob(args["knob_name"], args["value"])
```

或者用 `tool/utils.py` 提供的装饰器直接转换：

```python
from training.tool.utils import function_to_tool
set_knob_tool = function_to_tool(db_tools.set_knob)
```
